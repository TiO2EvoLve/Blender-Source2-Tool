|    功能    | 进度 | 备注 |
|:--------:|:--:|:--:|
|  单模型导出   | ✔️ | 无  |
|  多模型导出   | ✔️ | 无  |
| 单模型多材质导出 | ✔️ | 无  |
| 多模型多材质导出 | ✔️ | 无  |
|  自动生成碰撞  | ❌  | 无  |
